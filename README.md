# RTLGuard AI — Next-Gen Semiconductor RTL Verification Platform

RTLGuard AI is a production-ready Verilog/SystemVerilog RTL verification platform with AI-powered bug detection, testbench generation, coverage analysis, and visualization for VLSI engineers.

## Features

- **AI-Powered Bug Detection:** Automatically identify syntax errors, logic bugs, and style issues in RTL code.
- **Interactive Dashboard:** Modern UI for monitoring synthesis, testbench, and verification statistics.
- **Coverage Analysis:** Visualize testbench coverage metrics.
- **Code Editor:** Integrated Monaco Editor with syntax highlighting for Verilog/SystemVerilog.

## Technology Stack

- **Frontend:** React (via ESM), Tailwind CSS, Monaco Editor, Lucide Icons.
- **Backend:** Python Web Server (Flask/FastAPI based) for running analysis tasks.
